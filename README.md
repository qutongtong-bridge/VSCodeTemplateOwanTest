# VS Code Template - QR Marketing System

基于 VS Code 开发容器的项目模板，包含 QR 营销追溯系统。

## 主要功能
- VS Code 开发容器配置
- QR 营销追溯系统 (React + Ant Design)
- Claude Code CLI 集成
- Apidog MCP Server 集成

## 快速开始
1. 克隆项目
2. 在 VS Code 中打开
3. 重新构建容器
4. 运行 QR 营销系统：
   ```bash
   cd Qrmarketing
   npm install
   npm start
   ```
