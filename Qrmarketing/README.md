# QR Marketing 营销追溯系统

一个基于 React 和 Ant Design 的 QR 码营销追溯管理系统。

## 功能特性

- 产品类别管理
- 产品参数分组
- 产品参数设置
- 产品管理
- 响应式设计
- 现代化 UI 界面

## 技术栈

- React 18
- React Router v6
- Ant Design 5
- Axios
- Day.js

## 安装和运行

1. 安装依赖：
```bash
cd Qrmarketing
npm install
```

2. 启动开发服务器：
```bash
npm start
```

3. 构建生产版本：
```bash
npm build
```
